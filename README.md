# Diablo Mobile Clone

Minimal top-down ARPG for touch screens.

## Run locally

Open `index.html` in a mobile browser, or `python3 -m http.server 8080` then visit `http://localhost:8080`.

## GitHub Pages

1. Create repo `diablo-mobile-clone` on GitHub.

## Controls

- Left: virtual joystick (drag to move)
- Right: red ATTACK button (tap to attack enemies in range)
- Restart appears when HP reaches 0

## Notes

- Pure static files, no server needed.
- Touch controls prevent scrolling/zooming.
